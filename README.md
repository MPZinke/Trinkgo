# Trinkgo

A song based version of Bingo using Spotify Web App.


## Key Features
- Imports Spotify playlists to a DB for selecting song snippets.
- Use playlists to generate random Bingo cards as PDFs & download for events.
- Plays playlists songs snippets as games, tracking played snippets and songs left in playlist.
